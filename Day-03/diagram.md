# Linux Basics – Visual Diagram

## Linux File System Structure

/
├── bin        → basic system commands
├── home       → user files
│   └── user
│       ├── Documents
│       └── Downloads
├── etc        → configuration files
├── var        → logs and variable data
└── tmp        → temporary files

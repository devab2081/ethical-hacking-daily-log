## Ports and Services
```
Computer
│
├── Port 80 → Web service
├── Port 443 → Secure web service
├── Port 22 → Remote access
└── Other ports → Other services

```

## TCP vs UDP
```
TCP
│
├── Reliable
├── Slower
└── Used in web, email

UDP
│
├── Faster
├── Less reliable
└── Used in streaming, gaming

```
